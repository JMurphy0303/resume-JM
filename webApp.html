Foodie Finder
Description: A location-based app that helps users find nearby restaurants, view menus, and read reviews. Includes a personalized recommendation engine. Tech Stack:
  • Frontend: React, Tailwind CSS
  • Backend: Node.js, Express
  • Database: MongoDB
  • APIs: Google Maps API, Yelp Fusion
Fitness Coach Hub
Description: An interactive fitness app featuring custom workout plans, video tutorials, and progress tracking for users of all levels. Tech Stack:
  • Frontend: Vue.js, Vuetify
  • Backend: Larvel
  • Database: MySQL
  • Features: Live Chat with Coaches, Video Streaming
GreenShop E-Commerce
Description: An eco-friendly e-commerce platform offering sustainable products with options for carbon offset contributions at checkout. Tech Stack:
  • Frontend: Angular, Material Design
  • Backend: Django REST Framework
  • Database: PostgreSQL
  • Payment Gateway: Stripe
Travel Planner Plus
Description: A trip-planning app where users can create itineraries, share plans with friends, and get recommendations for activities and accommodations.
Tech Stack:
  • Frontend: Svelte, Tailwind CSS
  • Backend: Flask
  • Database: Firebase
  • APIs: Skyscanner, OpenWeather
LearnNow Online Academy
Description: A web-based learning platform offering interactive courses, quizzes, and certifications across various subjects. Tech Stack:

  • Database: SQLite

PetPals Social Network
