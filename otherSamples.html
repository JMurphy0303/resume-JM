Budget Buddy
Description: A personal finance management app that helps users track expenses, create budgets, and analyze spending patterns. Includes visual reports and alerts for overspending.

  • Backend: Spring Boot
  • Database: PostgreSQL
  • APIs: Plaid API
Eventify
Description:
A web app for creating and managing events. Users can send invitations, track RSVPs, and share event updates in real time.
  • Frontend: Angular, Material Design
  • Backend: Node.js, Express
  • Database: MongoDB
  • APIs: Twilio
ArtGallery Online

Description:
A virtual gallery platform for artists to showcase their work. Users can browse collections, read about the artists, and purchase artwork securely.
  • Frontend: Gatsby.js, Styled Components
  • Backend: Ruby on Rails
  • Database: SQLite
  • Features: Image optimization, Stripe for payments
